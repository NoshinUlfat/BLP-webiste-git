Motivation

Large-Language Models (LLMs) have revolutionised code generation for high-resource languages. Bangla, the world’s 6th most spoken language, is still under-explored. This shared task invites the community to fine-tune or adapt an LLM that converts Bangla instructions into correct Python code.

Who Can Participate?

Eligibility Open worldwide — BS, MS, PhD students or any independent researcher
Team Size Up to 3 members
Fees None
